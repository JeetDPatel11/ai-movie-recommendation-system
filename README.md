# 🎬 AI-Based Movie Recommendation System

An AI-powered movie recommendation system that suggests similar movies using **semantic similarity, genre overlap, ratings, and popularity metrics**.

The application is built using **Python, Streamlit, and Sentence Transformers**, and provides an interactive interface for discovering new movies.

---

# 🚀 Features

• Semantic movie similarity using AI embeddings
• Hybrid recommendation algorithm
• Genre-based similarity scoring
• Popularity and rating weighting
• Film industry filtering (Bollywood, Hollywood, Tollywood, etc.)
• Country-based filtering
• Movie posters and details using OMDb API
• Interactive web interface built with Streamlit

---

# 🧠 Recommendation Algorithm

The system uses **semantic embeddings generated by Sentence Transformers** to understand the meaning of movie descriptions.

Final recommendation score:

Final Score =
0.6 × Semantic Similarity

* 0.2 × Genre Similarity
* 0.1 × Rating Weight
* 0.1 × Popularity Weight

This hybrid scoring approach improves the quality and relevance of recommendations.
